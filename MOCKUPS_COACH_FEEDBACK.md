## Questions Answered:

### Q: Will this impact existing functionality?
**A**: No. Public view remains 100% unchanged. New sections only appear for logged-in parents.

### Q: Where does coach enter feedback?
**A**: Separate coach dashboard (`/auth/coach-dashboard.html`) - completely isolated from main page.

### Q: Do coaches see the main stats page?
**A**: Yes! Coaches can click "View Stats" for any swimmer to see the existing public view. They work in their own dashboard for feedback entry.

### Q: What if a parent doesn't want to see feedback?
**A**: They can collapse the "Coach Feedback" section (like any other section) or continue using the site without logging in.

### Q: Can public users still see everything they see now?
**A**: Yes! 100% unchanged experience for public/anonymous users.
