# Vihaan Swim Tracker

Track swim times, progress toward time standards, and meet preparation for Vihaan Huchchannavar.

## Live Links

### Main Dashboard
- **Progress Tracker**: https://yvh1223.github.io/vihaan-swim-tracker/

### Meet Prep Documents

### Practice Guides (in docs/)
- Daily Practice Focus
- Race Day Reminders
- Butterfly Practice Notes

## Features

- Track personal bests across all events (SCY/LCM)
- View progress toward USA Swimming time standards (B, BB, A, AA, AAA, AAAA)
- Visualize improvement trends with charts
- Generate meet-specific workout plans
- Parent observation guides

## Data Source

Swim times are stored in Supabase and pulled from USA Swimming results.

## Tech Stack

- HTML/CSS/JavaScript
- Supabase (database)
- GitHub Pages (hosting)
